# Anastasia-Nikolskaya.ru

Gentle guidance before and after childbirth.

Bilingual (RU/EN) Astro site for Anastasia Nikolskaya — prenatal and postpartum personal trainer in Samara.

## Stack

- [Astro](https://astro.build/) 5
- [Tailwind CSS](https://tailwindcss.com/) 4
- [Preact](https://preactjs.com/) (interactive islands)
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
